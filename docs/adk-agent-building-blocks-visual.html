What we should build next

A real stability_status_report() capability: one visible artifact showing checks, risks, allowed actions, blocked actions, and next safe move.

The useful trick: components stay boring and tested. Agents stay thin and opinionated. Once the components are solid, new agents become assemblies rather than reinventions. Less spaghetti, fewer tiny haunted puppets.
