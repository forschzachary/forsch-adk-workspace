# Social web entrypoint directory note

Purpose: expose the packaged social agent to `adk web` / `adk run` without duplicating agent logic.

Structure:

- `agent.py` - imports `root_agent` from `forsch.agent_social.agent` and aliases it as `agent` for ADK Web UI discovery.
- `root_agent.yaml` - editable ADK Web surface generated by the factory from `agent_specs/agents.yaml`.

Current organization rule: do not put business logic here. Edit the social graph in `/root/.hermes/workspace/adk/agents/social/` instead.
