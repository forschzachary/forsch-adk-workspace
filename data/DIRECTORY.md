# TBC data directory

This directory holds the TrinityCore TBC database and related data files.

- `schema.sql` — SQLite schema for items, creatures, loot, quests, dungeons, and players.
- `tbc_data.sqlite` — the live database (gitignored, generated by data import).
- `import_data.py` — script to download and import TrinityCore TBC dump (Phase 1, not yet written).

The database is read-only at runtime. The wow-guild agent queries it via the tools in `forsch.adk_components.tools.wow_tools`.
